## githeart

paintable github contribute table

![image](https://user-images.githubusercontent.com/48323786/130625675-945dcc00-dcc9-45d5-8319-29d8fd5c3b13.png)
 
 # how to use:
 - color select:
   -  1,2,3,4,5 key

 - clear:
   - c key
 - drawing mode mode on turn off
   - e key
 - print paint matrix
   - p key
